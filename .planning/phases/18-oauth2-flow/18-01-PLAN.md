       - `pub async fn get_auth_status(&self, plugin_name: &str) -> Result<String>` -- check if `access_token` credential exists: if yes and `expires_at` is in the future -> "connected"; if yes but expired/null -> "expired"; if no row -> "not_configured". No decryption needed.

<objective>
Implement the divergent subagent mode: spawn 2-3 parallel framings of the same problem with different system prompt perspectives, detect disagreements between their outputs, surface tensions as the valuable output (not forced consensus), and synthesize a mediator recommendation that acknowledges tradeoffs.

Purpose: Enable the agent to explore ambiguous problems from multiple angles simultaneously, surfacing productive disagreement rather than forcing premature consensus. This is the most novel capability in Phase 3.
Output: divergent.rs with DivergentSession, tension formatting, mediator prompt generation, and integration with existing CollaborationSession.
</objective>

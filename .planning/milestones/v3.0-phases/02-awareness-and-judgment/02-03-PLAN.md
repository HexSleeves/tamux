<objective>
Uncertainty quantification module: structural confidence scoring for goal plan steps, domain-specific escalation, operator-configurable thresholds, calibration feedback loop, and goal planner approval routing.

Purpose: The agent must express honest confidence grounded in structural evidence, not LLM self-assessment. This plan creates the confidence computation pipeline that combines 4 structural signals (tool success rate from awareness, episodic familiarity from embodied, blast radius from causal traces, approach novelty from counter-who) into HIGH/MEDIUM/LOW labels per plan step. Domain classification determines escalation behavior (Safety blocks on LOW, Research surfaces all). The calibration tracker learns from operator feedback over time.

Output: uncertainty/ module with 4 files, UncertaintyConfig on AgentConfig, goal plan step annotation, plan confidence gate, heartbeat trajectory integration.

    ```rust
    use crate::agent::explanation::{confidence_band, ConfidenceBand};
    use super::domains::{DomainClassification, DomainThresholds};

    pub struct ConfidenceSignals {
        pub tool_success_rate: f64,     // 0.0-1.0, from awareness window
        pub episodic_familiarity: f64,  // 0.0-1.0, from embodied compute_familiarity
        pub blast_radius_score: f64,    // 0.0-1.0, from causal_traces advisory
        pub approach_novelty: f64,      // 0.0-1.0, from counter_who tried_approaches
    }

    pub struct ConfidenceAssessment {
        pub band: ConfidenceBand,
        pub label: &'static str,  // "HIGH", "MEDIUM", "LOW"
        pub evidence: Vec<String>,
        pub domain: DomainClassification,
        pub should_block: bool,
    }

    /// Map ConfidenceBand to user-facing label (UNCR-01).
    /// Per locked decision: labels are HIGH/MEDIUM/LOW, not the 4-tier band names.
    pub fn confidence_label(band: ConfidenceBand) -> &'static str {
        match band {
            ConfidenceBand::Confident => "HIGH",
            ConfidenceBand::Likely => "MEDIUM",
            ConfidenceBand::Uncertain | ConfidenceBand::Guessing => "LOW",
        }
    }

    /// Convert blast radius advisory risk_level string to 0.0-1.0 score.
    pub fn blast_radius_to_score(risk_level: &str) -> f64 {
        match risk_level {
            "low" => 0.2,
            "medium" => 0.5,
            "high" => 0.9,
            _ => 0.5,
        }
    }

    /// Compute approach novelty (0.0-1.0) from counter_who tried_approaches.
    /// 0 prior attempts = 1.0 (completely novel), 5+ = 0.0 (proven pattern).
    pub fn approach_novelty_score(matching_prior_attempts: usize) -> f64 {
        let capped = matching_prior_attempts.min(5) as f64;
        1.0 - (capped / 5.0)
    }

    /// Compute confidence from structural signals only (UNCR-06).
    pub fn compute_step_confidence(
        signals: &ConfidenceSignals,
        domain: DomainClassification,
        thresholds: &DomainThresholds,
    ) -> ConfidenceAssessment {
        let score = 0.30 * signals.tool_success_rate
            + 0.25 * signals.episodic_familiarity
            + 0.25 * (1.0 - signals.blast_radius_score)
            + 0.20 * (1.0 - signals.approach_novelty);

        let band = confidence_band(score);
        let label = confidence_label(band);

        let mut evidence = Vec::new();
        if signals.tool_success_rate < 0.5 {
            evidence.push(format!("Low tool success rate: {:.0}%", signals.tool_success_rate * 100.0));
        }
        if signals.episodic_familiarity < 0.3 {
            evidence.push("Unfamiliar pattern (few similar past episodes)".to_string());
        }
        if signals.blast_radius_score > 0.7 {
            evidence.push("High blast radius: action could have wide impact".to_string());
        }
        if signals.approach_novelty > 0.7 {
            evidence.push("Novel approach (not previously attempted)".to_string());
        }

        let should_block = thresholds.should_block(domain, band);

        ConfidenceAssessment { band, label, evidence, domain, should_block }
    }
    ```

    Per locked decision: Safety domains block on LOW, Business warn on LOW, Research surface all levels.

    ```rust
    #[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
    #[serde(rename_all = "snake_case")]
    pub enum DomainClassification {
        Safety,       // destructive tools, system commands
        Reliability,  // deployment, config changes
        Business,     // standard file operations, code changes
        Research,     // web search, file reading, exploration
    }

    /// Classify a tool/action into a domain.
    pub fn classify_domain(tool_name: &str) -> DomainClassification {
        match tool_name {
            "execute_command" | "execute_managed_command" | "delete_file"
            | "kill_session" | "restart_session" => DomainClassification::Safety,
            "deploy" | "write_config" | "install_package" => DomainClassification::Reliability,
            "web_search" | "web_read" | "symbol_search" | "list_directory"
            | "list_files" | "search_files" => DomainClassification::Research,
            _ => DomainClassification::Business,
        }
    }

    #[derive(Debug, Clone, Serialize, Deserialize)]
    pub struct DomainThresholds {
        pub safety_block_below: ConfidenceBand,      // default: Likely (blocks LOW)
        pub reliability_block_below: ConfidenceBand,  // default: Uncertain (blocks LOW)
        pub business_block_below: ConfidenceBand,     // default: Guessing (warns LOW)
        pub research_block_below: ConfidenceBand,     // default: never blocks
    }
    ```
    Default thresholds per locked decision:

    - Reliability: block below Uncertain (i.e., block Guessing only)
    - Business: block below Guessing (warn on LOW, never block)

    ```rust
    /// Annotate each plan step with a confidence label [HIGH/MEDIUM/LOW] (UNCR-01).
    /// Uses structural signals only (UNCR-06): tool success rate from awareness,
    /// episodic familiarity from FTS5 hit count, blast radius from causal traces,
    /// approach novelty from counter-who.
    async fn annotate_plan_steps_with_confidence(
        &self,
        steps: &mut Vec<GoalRunStep>,
        goal_text: &str,
    ) {
        let config = self.config.read().await;
        if !config.uncertainty.enabled { return; }
        let thresholds = &config.uncertainty.domain_thresholds;
        drop(config);

        for step in steps.iter_mut() {
            // 1. Tool success rate from awareness window (AWAR-01 signal)
            let tool_success_rate = {
                let monitor = self.awareness.read().await;
                // Aggregate across all entity windows (returns 0.8 if no data)
                monitor.aggregate_short_term_success_rate()
            };

            // 2. Episodic familiarity: count FTS5 hits for step instructions
            let episodic_familiarity = {
                let query = &step.instructions;
                match self.retrieve_relevant_episodes(query, 5).await {
                    Ok(episodes) => super::embodied::dimensions::compute_familiarity(episodes.len()),
                    Err(_) => 0.5,  // default to moderate familiarity on error
                }
            };

            // 3. Blast radius: classify step by its expected tool usage
            let domain = super::uncertainty::domains::classify_domain(&step.kind);
            let blast_radius_score = match domain {
                super::uncertainty::domains::DomainClassification::Safety => 0.8,
                super::uncertainty::domains::DomainClassification::Reliability => 0.5,
                _ => 0.2,
            };

            // 4. Approach novelty: check counter-who for similar approaches
            let approach_novelty = {
                let store = self.episodic_store.read().await;
                let hash = super::episodic::counter_who::compute_approach_hash(
                    &step.kind, &step.instructions.chars().take(100).collect::<String>()
                );
                let matching = store.counter_who_state.tried_approaches.iter()
                    .filter(|a| a.approach_hash == hash)
                    .count();
                super::uncertainty::confidence::approach_novelty_score(matching)
            };

            let signals = super::uncertainty::confidence::ConfidenceSignals {
                tool_success_rate,
                episodic_familiarity,
                blast_radius_score,
                approach_novelty,
            };

            let assessment = super::uncertainty::confidence::compute_step_confidence(
                &signals, domain, thresholds,
            );

            // Prepend confidence label to step title (locked decision: "[HIGH] Step title")
            step.title = format!("[{}] {}", assessment.label, step.title);

            // Add confidence evidence to step instructions if not HIGH
            if assessment.label != "HIGH" && !assessment.evidence.is_empty() {
                step.instructions = format!(
                    "{}\n\nConfidence note: {}",
                    step.instructions,
                    assessment.evidence.join("; ")
                );
            }
        }
    }
    ```

    ```rust
    /// Check plan confidence and route to appropriate approval flow (UNCR-08).
    /// All HIGH -> proceed autonomously.
    /// Any MEDIUM -> inform operator via WorkflowNotice.
    /// Any LOW -> require operator approval before proceeding.
    async fn plan_confidence_gate(&self, goal_run: &GoalRun) -> PlanConfidenceAction {
        let config = self.config.read().await;
        if !config.uncertainty.enabled {
            return PlanConfidenceAction::Proceed;
        }
        drop(config);

        let mut has_medium = false;
        let mut has_low = false;
        let mut low_steps = Vec::new();

        for (i, step) in goal_run.steps.iter().enumerate() {
            if step.title.starts_with("[LOW]") {
                has_low = true;
                low_steps.push(format!("Step {}: {}", i + 1, step.title));
            } else if step.title.starts_with("[MEDIUM]") {
                has_medium = true;
            }
        }

        if has_low {
            let _ = self.event_tx.send(AgentEvent::ConfidenceWarning {
                thread_id: goal_run.thread_id.clone(),
                action_type: "plan_step".to_string(),
                band: "low".to_string(),
                evidence: low_steps.join("; "),
                domain: "mixed".to_string(),
                blocked: true,
            });
            PlanConfidenceAction::RequireApproval
        } else if has_medium {
            let _ = self.event_tx.send(AgentEvent::WorkflowNotice {
                thread_id: goal_run.thread_id.clone(),
                kind: "confidence".to_string(),
    ```

       - Use `sha2::Sha256` (already in daemon deps)

       `pub fn detect_repeated_approaches(tried: &[TriedApproach], threshold: usize) -> Option<String>`:
       - Group tried_approaches by approach_hash
       - For each group, count entries where outcome is Failure
       - If any group has >= threshold (default 3) failures, return Some with a suggestion string:
         `"Repeated failure detected: {description} has failed {count} times. Consider a different approach."`
       - If multiple groups qualify, return the one with the most failures
       - Return None if no group meets threshold

       `pub fn record_correction(state: &mut CounterWhoState, pattern: &str, now_ms: u64)`:
       - Search state.correction_patterns for matching pattern
       - If found, increment correction_count and update last_correction_at
       - If not found, push new CorrectionPattern { pattern, correction_count: 1, last_correction_at: now_ms }

       `pub fn prune_old_approaches(state: &mut CounterWhoState, now_ms: u64, max_age_days: u64, max_entries: usize)`:
       - Remove entries from tried_approaches where `now_ms - timestamp > max_age_days * 86400 * 1000`
       - If still > max_entries, keep only the most recent max_entries
       - Cap recent_changes at 50 entries (trim oldest)
